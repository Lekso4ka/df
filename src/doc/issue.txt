Страницы:
    *обязательные
        - главная
        - страница товаров (каталог)
        - страница одного товара
        - страница авторизации / регистрации
        - страница корзины
    дополнительные
        - личный кабинет
        - избранные товары
        - страница каталога по категориям
        - страница добавления товара
    статика
        - информация о доставке
        - информация о магазине

Элементы формы:
    Пользователь:
        name - text
        about - textarea
        avatar - link
        email - email
        password - password
    Товар:
        name - text
        price - number
        discount - select
        stock - number
        available ? isPublished - checkbox
        wight - text
        description - textarea
        pictures - link
    Отзыв:
        name - text
        city - text
        text - textarea
        rating - number
    Поиск:
        text / search

Фильтры в каталоге:
    По тегам
    Только со скидкой
    Только в наличии
    По цене от-до
    Поиск по названию товара
    По постащику (Автор) - id тех авторов, которые добавили все товары, не относящиеся к собакам (добавить их в BlackList)
